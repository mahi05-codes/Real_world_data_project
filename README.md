
# 🛒 Retail Sales Analysis and Sales Prediction

## 📌 Overview

This project demonstrates the complete data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, predictive modeling, and model evaluation using Python in Google Colab.

## 🎯 Objectives

* Analyze retail sales data to identify business trends.
* Perform data cleaning and preprocessing.
* Visualize sales, profit, and customer insights.
* Build a machine learning model to predict sales.
* Evaluate the model using performance metrics.

## 📂 Dataset

The project uses a retail sales dataset containing:

* Order Details
* Customer Information
* Product Category
* Region
* Sales
* Quantity
* Discount
* Profit
* Shipping Details

## 🛠️ Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Feature Engineering
6. Data Visualization
7. Machine Learning Model (Random Forest Regressor)
8. Model Evaluation
9. Business Insights

## 📈 Visualizations

* Sales by Category
* Profit by Region
* Sales Distribution
* Correlation Heatmap
* Sales vs Profit
* Monthly Sales Trend
* Feature Importance

## 🤖 Machine Learning

**Model Used:**

* Random Forest Regressor

**Evaluation Metrics:**

* R² Score
* Root Mean Squared Error (RMSE)

## 📌 Conclusion

This project demonstrates an end-to-end data science workflow by analyzing retail sales data, extracting meaningful insights, and building a machine learning model to predict sales. It showcases practical skills in data preprocessing, visualization, and predictive analytics using Python.
